# CrimsonDesert-HUD-Toggle
A Crimson Desert UI mod 

Adding the source here so that people can build it themselves if they're curious or have any concerns about ASI mods containing malicious code on Nexus Mods. 


1. Open Visual Studio 2022
2. Create new project: Dynamic-Link Library (DLL)
3. Replace dllmain.cpp with the one from this repo
4. Build in Release x64
